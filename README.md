# Rage balancer

## Installation && Build

```
# Install packages
npm install
```

```
# release build
npm run build 
```

```
# development mode
npm run dev
```

![UI](https://github.com/effus/rage-balancer/raw/master/wiki/2019-11-08_113937.png)

## Config

`src\config.yml`

```yml
window:
  weight: 500
  height: 400
  resizable: true
  maximize: false
  devTools: false
locale: en
```

Allowed locales: `en`, `ru`
